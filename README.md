## Что делает программа?

1. Парсит аргументы командной строки
2. Запускает чат в котором можно ввести четырехмерный вектор 2 на 2 на 2 на 2, тип введенный пользователем из поддерживающихся
3. Вывод вектора
4. Ввод строкового псевдонима (просто записывает значение в переменную)
